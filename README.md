# CRUD de Usuarios - Java + MySQL

## Descripción
Aplicación desarrollada en Java que permite gestionar usuarios (crear, listar, actualizar y eliminar) conectándose a una base de datos MySQL mediante JDBC.

## Tecnologías usadas
- Java
- MySQL
- JDBC
- NetBeans

## Funcionalidades
- Insertar usuarios
- Listar usuarios
- Actualizar usuarios
- Eliminar usuarios

## Estructura del proyecto
- conexion: conexión a la base de datos
- dao: lógica de acceso a datos
- modelo: clases del sistema
- main: ejecución del programa

## Base de datos
Nombre: mi_base  
Tabla: usuarios  

Campos:
- id (INT, AUTO_INCREMENT)
- nombre (VARCHAR)
- correo (VARCHAR)
